# GameMechanics Trainer

브라우저에서 바로 하는 게임 메카닉 훈련 웹 앱 (MVP).

## 실행

```bash
npm install
npm run dev
```

http://localhost:3000 접속.

## Phase 1 MVP 상태

- [x] 레이아웃 (사이드바 + 헤더, 다크 테마)
- [x] Aim Trainer (난이도 5단계, 30초 라운드, 결과 카드, localStorage 기록)
- [ ] Dodge Trainer — Coming Soon
- [ ] Reaction Speed Test — Coming Soon
- [ ] Rhythm & Timing — Coming Soon
- [x] 통계 페이지 (Aim 히스토리)
